## Uso de plantillas
En esta carpeta encontrarás la plantilla `.tex` para elaborar tu libro de códigos con el estilo y logo del Laboratorio de Métodos, 
así como un archivo de ejemplo para guiar su redacción.

**Nota:** Recuerda guardar la plantilla y la imagen en el mismo directorio de trabajo que tienes enlazado 
en Markdown para que puedas utilizarlas.
